Windows和Ubuntu双系统时间同步

发表于 2019-09-26

Windows 10 和 Ubuntu 双系统，安装后由于两个系统时间设置方式不一致，所以需要进行同步，才能解决切换系统后，时间不准确的问题。

在Ubuntu下进行以下操作， Ctrl + Alt + T 调出 Terminal, 输入以下命令。

安装ntpdate：

sudo apt-get install ntpdate

设置校正服务器：

sudo ntpdate time.windows.com

设置硬件时间为本地时间：

sudo hwclock --localtime --systohc

重启，切换到windows下查看结果。
